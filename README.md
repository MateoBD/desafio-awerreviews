# desafio-awerreviews
Desafio Inicial Full Stack para Awer Reviews

Objetivo
Crear una aplicación simple de Lista de Tareas que permita:
1. Frontend (React): Mostrar una lista de tareas y agregar nuevas
tareas.
2. Backend (Spring Boot): Exponer una API REST con operaciones básicas
(GET y POST).
3. Base de Datos (MySQL): Guardar las tareas en una tabla.
